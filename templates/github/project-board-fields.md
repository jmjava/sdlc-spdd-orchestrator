# GitHub Project Board Fields

Suggested custom fields for tracking SDLC-SPDD work:

| Field | Type | Values |
|-------|------|--------|
| Work ID | Text | FEAT-001-example |
| Canvas Status | Single select | Draft, Ready For Coding, In Progress, In Review, Done |
| Readiness | Single select | Ready For Coding, Needs Clarification, Needs Redesign, Blocked |
| Review Result | Single select | Approved, Approved With Notes, Changes Requested, Blocked |
| Stack | Multi select | Java, Spring Boot, Python, Node, Kubernetes, Tekton |

1. Backlog
