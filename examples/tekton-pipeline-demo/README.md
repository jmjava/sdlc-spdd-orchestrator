# Tekton Pipeline Demo

Lightweight example layout for pipeline work using SDLC-SPDD.

## Suggested Flow

1. Capture pipeline change requirement under `requirements/`
2. Plan with `/sdlc-spdd-plan`
3. Apply `templates/stack-rules/tekton.md` during architect review
4. Implement one pipeline operation at a time

Folders mirror the Spring Boot example: `requirements/`, `agent-context/features/`, `spdd/canvas/`.
