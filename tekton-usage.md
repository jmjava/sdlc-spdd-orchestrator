# Tekton Usage

Use SDLC-SPDD for pipeline and task changes the same way as application features:

1. Capture the requirement
2. Plan with a REASONS Canvas
3. Architect review with `templates/stack-rules/tekton.md`
4. Implement one pipeline change at a time
5. Review, retro, and sync

See `examples/tekton-pipeline-demo/` for a lightweight demo layout.
