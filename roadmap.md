# Roadmap

## MVP (current)

- Repository structure, templates, Cursor commands
- Init, detect, validate, and sync scripts
- Spring Boot example workflow
- GitHub issue/PR templates and canvas validation workflow

## Post-MVP

- Lightweight CLI wrapper
- OpenSPDD command compatibility
- SDLC Agents memory import/export
- GitHub issue/PR generation from canvas
- Tekton validation pipeline
- ArchUnit integration for Java
- Project-specific rule packs
- Multi-tool command generation (Claude Code, Codex, OpenCode)
- Feature status UI and metadata index
- Integrations with tekton-dag and CourseForge workflows
