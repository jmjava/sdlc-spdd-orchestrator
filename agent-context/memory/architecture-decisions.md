### ADR-001: SDLC owns process, OpenSPDD owns artifacts

- Date: 2026-05-23
- Status: Accepted
- Context: Two upstream frameworks inspire this project with different strengths.
- Decision: Use SDLC agent roles for lifecycle phases and REASONS Canvas for design contracts.
- Consequences: Commands and templates enforce role separation and explicit contracts.
