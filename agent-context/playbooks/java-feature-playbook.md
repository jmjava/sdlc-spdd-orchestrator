# Java Feature Playbook

1. Run `/sdlc-spdd-init` if the project is not initialized.
2. Capture the requirement under `requirements/`.
3. Run `/sdlc-spdd-plan` against the requirement.
4. Run `/sdlc-spdd-architect` until readiness is **Ready For Coding**.
5. Implement one operation at a time with `/sdlc-spdd-code`.
6. Run `/sdlc-spdd-review` after each meaningful change set.
7. Run `/sdlc-spdd-retro` when the work is complete.
8. Run `/sdlc-spdd-sync` to reconcile canvas and code.

Apply `templates/stack-rules/java-spring-boot.md` during planning and coding.
